# 版本管理
## node版本管理

- 安装NVM
`curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.38.0/install.sh | bash ` 

nvm 增删改查
```bash
nvm ls
nvm install <version>
nvm uninstall <version> 
# 切换node版本
nvm alias default <version> 
```
